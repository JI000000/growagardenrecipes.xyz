# Grow a Garden Recipes - Complete Roblox Cooking Guide

一个现代化的Roblox Grow a Garden食谱网站，提供完整的烹饪指南和详细的步骤说明。

## 🚀 项目特点

- **现代化设计**: 使用Tailwind CSS和现代UI组件
- **响应式布局**: 完美适配桌面和移动设备
- **SEO优化**: 完整的元数据和结构化数据
- **动画效果**: 流畅的交互动画和过渡效果
- **完整内容**: 涵盖所有Grow a Garden食谱

## 📁 项目结构

```
src/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # 根布局
│   ├── page.tsx           # 首页
│   ├── recipes/           # 食谱页面
│   ├── search/            # 搜索页面
│   └── globals.css        # 全局样式
├── components/            # React组件
│   ├── layout/           # 布局组件
│   ├── ui/               # UI组件
│   └── search/           # 搜索组件
├── data/                 # 数据文件
├── hooks/                # 自定义Hooks
├── types/                # TypeScript类型
└── utils/                # 工具函数
```

## 🎨 设计系统

- **颜色主题**: 绿色和蓝色渐变主题
- **字体**: Inter字体家族
- **动画**: CSS动画和Framer Motion
- **组件**: 可重用的UI组件库

## 🛠️ 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **语言**: TypeScript
- **图标**: Lucide React

## 📱 页面结构

### 首页 (/)

- 英雄区域展示
- 热门食谱展示

- 制作指南链接
- 关于我们信息

### 食谱页面 (/recipes)

- 所有食谱分类展示

- 食谱搜索功能
- 按难度筛选
- 快速搜索标签

### 具体食谱页面 (/recipes/[slug])

- 详细的制作步骤

- 相关食谱推荐

## 🚀 快速开始

1. 安装依赖:

```bash
npm install
```

2. 启动开发服务器:

```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 📦 构建和部署

```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
